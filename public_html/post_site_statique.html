M104 Développer des sites web statiques

Dans cet article, vous allez découvrir ce que vous devez savoir ce module!

Découvrir

اهلا و مرحبا, فهاد البوسط غانتكلم على module ديال site web statique لي الصراحة مهم بزااف وهو المدخل ديال اي مبرمج لعالم الويب, فشنو غاتقرا فيه وكيفاش تمشي فيه مزيان ؟ ول حاجة module ديال "site web statique" غاتقرا فيه 3 حوايج : - HTML - CSS - BOOTSTRAP

أولا.HTML : هي لغة تانصاوبو بيها صفحات ديال الموقع. هي اختصار لــ hypertext Markup LanguageH فهو لتايقادو بيه الهيكل ديال الموقع و وضع كل العناصر لبغيناها تبان في الموقع ديالنا يعني العناوين، الفقرات،الصور،الفيديوهات ،القوائم الخ …


ثانيا CSS : هي لغة تاتخلينا نديرو style لداكشي لصاوبنا ب HTML مثلا بغينا العنوان اكون بلون أحمر والتصاور كبار و نقادو التصميم ديال الموقع هو اختصار لــ Cascading Style Sheets

اي انها تاتخلينا تحكم فكل ما يتعلق ب الحجم و تحديد الالوان, الخطوط,الهوامش والمسافات لعناصر الموقع .


ثالثا BOOTSTRAP : نفترضو صوبنا site بلغة HTML و درنا style ب CSS كلشي مزيان ولكن الا دخلتي للموقع من شي جهاز آخر عندو شاشة اكبر او أصغر بحال التلفون او Tablette، الموقع غايتخربق كاامل.
حنا بغينا نصاوبو موقع اخدم فجميع أنواع الشاشات ومتوافق من جميع الأحجام، ولكن كيفاش؟ فهادشي علاش خاصنا BOOTSTRAP .

لذلك ف bootstrap تايتعتابر framework ديال css من ناحية تيساعدنا على انشاء موقع او تطبيق متوافق مع جميع الشاشات ومن ناحية تايعطينا des styles موجودين تايساهلو علينا و تانختاصرو الوقت و هو مهم وضروري حيتاش كل المواقع و التطبيقات تايكونو responsive .

فهادشي لكاين فهاد البوسط حاولت نشرح للطلبة الجدد شنو لغايقراو فهاد module باختصار و تانتمنى تكونو ستافدتوا ,شكرااا !

M106 Manipuler des bases de données

Dans cet article, vous allez découvrir ce que vous devez savoir ce module!

Découvrir

السلام عليكم فهاد المقال غادي نتكلم على module ديال SGBD, اولا نفهمو بعدا بعض المفاهيم ,شناهي base de données ؟ وفين نقدرو نحتاجوها وشنو غانقرا بالضبط هادشي لغانشوفو فهاد البوسط.

بكل بساطة ملي تانسمعو قاعدة البيانات اي واحد البلاصة ديال التخزين تانخبيوا فيها المعلومات , مزيااان و بالنسبة SGBD هي اختصار ل Système Gestion Base de Données .أي نظام تسيير قواعد البيانات .

مثلا نفترضوا عندي بزاف ديال les données و بغيت نخزنهم و نتعامل معاهم فلدلك غايحصني نظام فين ندير هادشي ,هاد النظام او logiciel تايعطيني الامكانية باش نصاوب base de données و نخزن فيها المعلومات و نتعامل معها بمختلف العمليات سواء الوصول ليها او التغيير ,الخدف ,,,

فهاد النظام SGBD فين مجموعة ديال انواع مثلا : MYSQL - Oracle - SQL_Server ... كلهم عندهم نفس الدور,تايعطيونا واحد interface فين نقدرو ندخلو المعطيات و نبحثو على شي حاجة الى بغيناها اي الامكانية ديال تسيير les données . جميع هاد ل SGBD عندهم واحد اللغة مشتركة تاتخلينا حنايا كمبرمجين نتواصلو مع قاعدة البيانات و نوصلو للمعلومات لبغينا و نقدرو ندخلو معلومات جديدة و ندير جميع العمليات بدون مانحتاجو ل GUI ,فقط ف CMD,تانديرو les requetes او الطلبات لقاعدة البيانات ديالي.

هاد اللغة المشهورة خاصة بقواعد البيانات سميتها SQL ,تانستعاملوها مع غاع RELATIONNEL SGBD سواء MYSQL ,ORACLE,SQL-SERVER… حيتاش كاينين SGBD NON RELATIONNEL لمغانستعملوش معاهم langage SQL بحال MONGODB لغادين تقراوه فالعام الثاني. فهادشي لكاين فهاد البوسط حاولت نشرح للطلبة الجدد شنو لغايقراو فهاد module باختصار و تانتمنى تكونو ستافدتوا ,شكرااا !
